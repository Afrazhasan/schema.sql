# schema.sql

football.db database schema (tables, views, etc.) scripts (in SQL) and documentation



The `football.db` includes the following tables:

* teams
* games
* events
    * events_teams (join table)
* rounds
* groups
    * groups_teams (join table)
* leagues
* seasons
* players
* rosters (join table)
* goals
